# Streamlit app for Shopper Spectrum

Run the app locally:

```bash
pip install -r requirements.txt
streamlit run streamlit_app/app.py
```

Pages:
- Home — Overview metrics and quick actions
- Customer Segmentation — View RFM clusters and sample customers
- Product Recommendation — Interactive item-based recommendations
# streamlit_app/

Contains the Streamlit demo and assets to showcase segmentation and recommendations.

Run locally with:
```
streamlit run streamlit_app/app.py
```
